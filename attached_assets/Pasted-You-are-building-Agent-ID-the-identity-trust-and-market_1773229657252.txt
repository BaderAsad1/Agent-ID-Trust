You are building Agent ID — the identity, trust, and marketplace layer for AI agents.
Think DNS + OAuth + Stripe Atlas + Upwork, but purpose-built for autonomous AI agents.

Your job in this prompt is to build the complete frontend: marketing site, onboarding
flow, agent dashboard, public profile page, marketplace, and .agent domain management UI.

Use realistic mock data throughout. Mark every future API call with a clear TODO comment
showing the expected data shape. Do NOT wire real APIs yet. Focus entirely on making this
look and feel like a category-defining product.
